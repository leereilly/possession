# Posession

Possessive apostrophes are hard. Especially if they're for singular nouns ending in `s` or `ez`.

```ruby
"Chris".possessive(:ap)
# => "Chris'"

"Chris".possession(:chicago)
# => "Chris's"

"Chris".possession(:oxford)
# => "Chris's"

"Chris".possession(:gollum)
# => "Chrisses's"
```
